Github Cheat Sheet


This jargon is from the markdown syntax.

`# This is an <h1> tag`

`###### This is an <h6> tag`

`*This will create italic text*`
`_This will also be italic_`
`**This will bold text**`
`__This will also bold text__`

`*You **can** combine them too*`
